# **App Name**: JaguarSpeak

## Core Features:

- Multilingual Phrase Display: Display key phrases in Spanish, English, French, and Italian, catering to diverse visitor demographics.
- Phonetic Transcription: Show phonetic transcription (IPA) of phrases in all languages to aid pronunciation.
- Audio Pronunciation: Enable users to listen to audio pronunciations of phrases recorded by native speakers.
- Sectioned Phrase Organization: Categorize phrases by park area (Reception, Zip-lining, etc.) for easy access and context.
- Practice Mode: Provide an educational tool where service providers, like student interns, can practice and rehearse phrases.

## Style Guidelines:

- Primary color: Forest green (#388E3C), reflecting the natural environment of the Jaguar Ecotourism Park.
- Background color: Light green (#E8F5E9), a desaturated shade of the primary color that maintains a bright theme.
- Accent color: Yellow-Green (#827717), offering contrast and guiding user attention to key interactive elements.
- Body and headline font: 'PT Sans', a clear and modern sans-serif, readable for all languages featured in the application.
- Use nature-inspired icons to represent each section, visually linking phrases to park activities.
- Intuitive layout with clear visual cues (icons, flags) to facilitate usage for users with varying tech experience.
- Gentle transitions and feedback animations upon phrase selection and playback for a smooth user experience.
